How to Connect

  1. Launch Minecraft (version 1.12.2)
  2. Go to "Multiplayer"
  3. Click "Add Server"
  4. Enter Panic Shack for the "Server Name"
  5. Enter panic-shack.hallada.net for the "Server Address"
  6. Click "Done"
  7. You should now see the server in the list and be able to join it!
